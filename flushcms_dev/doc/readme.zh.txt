				
				FlushCMS

FlushCMS 是一套基于PHP+Apache+Mysql的内容管理系统. 
如果您想快速建立您自己的web站点,
如果您想简单直观地管理您的web站点,
请尝试一下想使用本 web　软件系统!

它采用向导，引领您一步步建立您的站点。
在预览模式下，您想更改某处内容，只需在您要更改的位置上，点击鼠标右键，您就可轻松地更改该位置内容。

我们希望此软件对您有用，谢谢！
