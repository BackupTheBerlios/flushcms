			
			FlushCMS

Thank you for downloading FlushCMS,This INSTALL will guide through the basics of installation and operation of
FlushCMS. Please ensure you read this and the accompanying documentation fully before proceeding with the 
installation.

===============
1. Installing FlushCMS
===============
 .Get FlushCMS from http://flushcms.518ic.com
 .tar -xvf flushcms.x.x.x.xxx.tar.gz
 .cp flushcms /usr/local/apache2/htdoc/flushcms (copy flushcms file to your publich html directory)
 .Import data file Sql/flushcms.mysql.sql to your mysql database
 .Edit Config/Config.php

	$DB_Host = "localhost";//change your database host
	$DB_UserName = "root";//change your database user name
	$DB_PassWord = "test1234";//change your database user password
	$DB_Name = "flushcms";//change your database name
 .Login as administrator http://your.flushcms.com/Admin
	user name:admin
	pass word:admin
